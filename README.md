# GPT research

This repo is mainly my hand-on experiments based on Andrej Karpathy tutorial videos.

## Installation

```
pip install -e .
```

## Usage

This python package is called `tinygpt` and use cli to perform train, val, and generate. Checkout `exps/train.sh` for more.
